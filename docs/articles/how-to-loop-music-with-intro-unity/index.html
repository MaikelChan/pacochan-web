How to loop music with intro in Unity

Introduction
#

Those of you who have a bit of experience using Unity, probably know that to play an audio loop you just need to create an AudioSource, assign an AudioClip and check the box that says Loop. Easy, isn’t it? But that’s not exactly what I’m here to talk about. I’m talking about when you have a music file that contains an intro. The idea is to play the intro once and then the rest of the song in a loop, something very typical in videogames.

If you look for tutorials, in forums or blogs, you will see people always explaining one of these two methods:

  • Separate the intro and the rest of the song into two different files, play the intro only once, and then loop the rest.
  • Write some code that gets every frame the number of the sample that is currently playing, detects when the audio is about reach the end, and sets a sample number that corresponds to just when the intro ends so that Unity rewinds and plays that part again.

Both methods are not perfect: they may have small audio cuts, or involve having extra code processing each frame.

So what if I told you that there is a better, perfect and efficient way to do this, that apparently nobody knows about? I assure you that I have searched many tutorials, forum threads and blogs, and I have never seen anyone comment on this method, I have not even seen it documented in the official Unity documentation! But one day I discovered it by chance.

How it’s done
#

The key thing that nobody knows is that Unity automatically detects loop points in WAV files. Many won’t know that this format supports loop points, since most audio programs don’t support this feature. But there is one that I have always used for these matters, that works very well, is free and not very well known: Wavosaur. The only drawback is that it is Windows exclusive and I don’t know of similar alternatives for other operating systems.

With this program you can open an audio, set loop points and save it as WAV:

  1. Open the audio file in Wavosaur.
  2. Select approximately the area of the audio that you want to loop.
  3. Go to the Tools/Loop/Create loop points menu. This will create the loop points, indicated in yellow with the texts Loop start and Loop end.

  1. Zoom in and drag the loop points precisely so that when the playback point returns to the start of the loop, it does so without any noticeable cuts or clicks.
  2. Check that the loop is playing correctly by double-clicking inside the loop area to select it, activate the little Loop button, and press play. The audio will repeat at the points you have marked.

  1. Once you are satisfied, save the file in WAV format by going to the File/Save As... menu.
  2. Import the audio into Unity, place it in an AudioSource, activate Loop and Unity will be able to play the audio keeping the loop points without the need to do any weird tricks or write a single line of code.

Conclusion
#

I discovered this little trick by chance, one time that I imported an audio that already had some loop points that I made years ago for use in another program. Unity didn’t play the audio from start to finish and at first I didn’t know why, until I decided to check if the audio had loop points (I didn’t remember I put them in), and I confirmed that it did and they matched how Unity played the audio.

So I hope this is helpful. I still can’t believe I haven’t been able to find anyone else who knows about this… but there has to be someone out there, right?
